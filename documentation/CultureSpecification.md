## Culture Module: Complete Specification

Culture represents the herd's **evolving identity, values, and anti-Necrocratic Dragon strategies** that emerge from survival → stabilization → thriving progression. Culture shapes rituals, diplomacy, combat approaches, and foal initiation rites. It emerges organically from role assignments, journal events, and world interactions.

## Core Components

### 1. Culture Traits (0-100 Development)
```
Each trait develops through repeated behaviors:
- River Culture: Water rituals, cleansing pollution (Priestess + Riverlands stay)
- Solar Culture: Dawn rites, solar item crafting (Storytellers + Solarbloom)  
- Mountain Culture: Endurance ceremonies, malnutrition resistance (Sky Burials)
- Plains Culture: Speed rituals, Travel optimization (Cartographers + long Trot/Canter)
- Rebel Culture: Anti-Esroh tactics (Gladiator victories + Esroh scrap crafting)
```

**Trait gain**: +1 per relevant role-day (Priestess in Riverlands = +River daily). Max 3 traits @ 100.

### 2. Herd Culture Level
```
**Survival (0-20 traits total)**: Prey animal, no rituals
**Stabilization (21-50)**: Basic rites emerge
**Culture (51-100)**: Distinct identity forms  
**Legendary (100+)**: World-shaping vs Necrocratic Dragon
```

### 3. Cultural Memory (Journal-Driven)
```
**Living Events**: Horses who remember → Storytellers spread
"First Esroh victory" (5 horses remember) → Rebel Culture +1
"100th Sky Burial" → Mountain Culture +5
```
**Lost Events** (no rememberers): Cultural collapse risk (-10 total traits)

```

• River: Water blessing (+300 Healer/Social XP)

Rite cost: 5x biome materials + 1 Priestess day
```

```

```
**Stay 7+ days**: Local culture influence (+1 their trait to your herd)

## Cultural Progression Triggers

### Survival Phase → Stabilization
```
Trigger: First 3 culture traits reach 20+
Effects:
✅ First Priest/ess viable (Day 8 efficiency)
✅ Basic rites unlock (materials → +10 morale)
✅ Journal events generate culture XP
✅ First initiation rite available
```

### Stabilization → Culture Phase
```
Trigger: Any trait hits 50 + 2 Storytellers active
Effects:
✅ Culture name emerges: "Riverborn Herd"
✅ Unique rite recipes (culture-specific)
✅ Diplomacy screen unlocks (trade/joiners)
✅ Rebel Culture → Esroh fight tactics
```

### Culture → Legendary
```
Trigger: 100+ total traits + "Necrocratic Dragon roar" event
Effects:
✅ World map special nodes (Dragon facilities)
✅ Multi-trait rites (+1k foal XP)
✅ Allied herds join anti-Dragon coalition
✅ Unique journal: "Herd of Legend"
```
